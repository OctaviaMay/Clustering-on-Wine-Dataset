This clustering task is performed on the Wine dataset, which contains the results of a chemical analysis of wines grown in a specific region of Italy.
The dataset is available at https://archive.ics.uci.edu/ml/datasets/wine

Using scikit-learn, clustering experiments are conducted with different values of k, and the optimal value of k is determined by selecting the most relevant features
using the Recursive Feature Elimination (RFE) technique.

K-means clustering and the DBSCAN algorithm are applied to perform a comparative clustering analysis.
After the clustering task, anomaly detection is carried out to identify and remove the most significant outliers.
